# Contributing to Airbnb Manager

Thanks for your interest in contributing! This is an open-source Airbnb property management platform, and we welcome contributions of all kinds.

## How to Contribute

### 1. Fork & Clone

```bash
git clone https://github.com/YOUR_USERNAME/airbnb-manager.git
cd airbnb-manager
```

### 2. Set Up Locally

```bash
make start
# Backend API: http://localhost:3001
# Dashboard: http://localhost:3001
```

Or with Docker:
```bash
docker-compose up
```

### 3. Make Your Changes

- Create a branch: `git checkout -b feature/your-feature`
- Make your changes
- Test locally
- Commit with clear messages

### 4. Open a Pull Request

- Push your branch
- Open a PR against `main`
- Describe what you changed and why

## What We Need Help With

- 🎨 **UI/UX improvements** — Better mobile experience, accessibility, animations
- 🌍 **i18n** — Translations (Spanish, Portuguese, French, etc.)
- 📊 **Analytics** — Better charts, revenue forecasting, occupancy trends
- 🤖 **AI features** — Smarter guest response templates, pricing optimization
- 🔌 **Integrations** — Booking.com, VRBO, Google Calendar sync
- 📱 **Mobile app** — React Native or Flutter companion app
- 🧪 **Testing** — Unit tests, E2E tests, API tests
- 📝 **Documentation** — Setup guides, API docs, user manual

## Code Style

- **Frontend:** Vanilla JS (no framework required), CSS custom properties
- **Backend:** Node.js + Express, SQLite
- **Keep it simple:** This project values simplicity and readability

Look for issues labeled `good first issue` — these are beginner-friendly tasks.
